6.编辑距离

思路：

  • f[i][j]代表长度为i的s编辑成长度为j的t所需的最小编辑次数；

  • 分成几种情况考虑：

    • 如果s[i]==s[j], f[i][j] = f[i-1][j-1];

    • 其余的是不相等的情况：

      • 增，f[i][j] = f[i-1][j] + 1;

      • 删，f[i][j] = f[i][j-1] + 1;

      • 变，f[i][j] = f[i-1][j-1] + 1;

    • 以上三个保留一个最小值即可。
