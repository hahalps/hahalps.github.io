记录一下填的第一坑，

这些知识点学至yxc大佬，orz

本章主要讲的有：

  • 欧几里得算法
  • 扩展欧几里得算法
  • 分解质因数
  • 线性筛
  • 欧拉函数
  • 快速幂

1,欧几里得算法
运用的是辗转相除法，不懂的可以简单模拟一下

2,扩展欧几里得算法
ax+by= gcd(a, b)
就是进行gcd的时候，同时改变a,b的系数

3,分解质因数 约数 欧拉数
假设 n = p1^a1 * p2 ^a2 * …. * pn^an
phi(n) = n*(1-1/p1)*(1-1/p2)* … * (1-1/pn)
处理细节 把n乘进去左边的括号里面

4，线性筛
有个剪枝的地方记下：当发现当前在筛的质数不是这个数的质因数当中最小的，就break
线性筛之所以是线性时间， 是因为保证每个数是被自己最小的质因数筛掉的。
